## Install and first start

1. `npm install` or `yarn add`
2. `cd api`
3.  `json-server --watch db.json --port 3001`
4. `cd ..`
5. `npm start` or `yarn start`
6. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
